# Design — Recollection: Same Age

## Physical scene

A person is alone late at night, holding an old family photo and realizing their parent once stood at this exact age. The interface should feel like a precise beam of festival light opening through a midnight room: intimate, vivid, and slightly unreal, never dusty or scrapbook-like.

## Color system

- Canvas: near-black indigo, `oklch(15% 0.035 300)`
- Surface: midnight plum, `oklch(20% 0.045 332)`
- Primary action: luminous saffron, `oklch(80% 0.16 82)`
- Living signal: electric fuchsia, `oklch(65% 0.22 350)`
- Memory signal: pale apricot, `oklch(92% 0.05 70)`
- Ink: deep aubergine on light surfaces, pale lilac on dark surfaces

The brand uses a committed palette: midnight carries the experience; saffron indicates an action or source of truth; fuchsia is the time-bridge signal. No beige paper surfaces.

## Type

Use a high-contrast serif for moments of emotional recognition and a clean system sans for every task, label, control, and form. Product controls remain compact and familiar; display type appears only for the age-bridge narrative.

## Signature motifs

- Orbital age rings and a central time lens
- A horizontal “Then / Now / Forward” timeline
- Source labels that read as part of the experience: Original / Spoken / Imagined
- Saturated light flares that move only when they explain a state change

## UX rules

- One decisive action per screen.
- The first thing a user does is choose an age bridge, not create a generic memory.
- Never use a card grid where a timeline, portal, or split-time composition tells the story more directly.
- Every video-generation control names the synthetic boundary and requires an intentional click.
- Respect reduced motion; all portal and orbit movement has an instant static equivalent.
